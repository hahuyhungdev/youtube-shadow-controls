# Browser Extensions

A repository containing custom web/browser extensions developed for language learning and shadowing practice.

## Extensions Included

### 1. [YouTube Shadow Controls](./youtube-shadow-controls/)
A keyboard-first extension for language shadowing practice on YouTube.
* **Key Features**: Direct playback rate adjustments, A-B looping point control, and aggregated action toasts.
* **Read more**: [YouTube Shadow Controls README](./youtube-shadow-controls/README.md)

### 2. [TikTok Shadow Controls](./tiktok-shadow-controls/)
A keyboard-first extension optimized for language shadowing practice on TikTok.
* **Key Features**: Viewport-aware active video selection (automatically locks onto the video you are looking at in the scroll feed), looping, rate controls, and styling matched to the TikTok brand.
* **Read more**: [TikTok Shadow Controls README](./tiktok-shadow-controls/README.md)
